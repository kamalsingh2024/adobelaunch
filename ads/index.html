AD Debugger Tool

Built and maintained by Kamal D Singh

Step 1 — Install

  1. Show your bookmarks bar: Ctrl+Shift+B (Windows) / Cmd+Shift+B (Mac)
  2. Drag the button below into your bookmarks bar

⚡ AD Debugger
Can't drag? Right-click the button → Copy Link → manually add a bookmark and paste as the URL.

Step 2 — How to Use

  1. Go to AFL Network website (or the relevant page you're testing)
  2. Once the page has loaded, click ⚡ AD Debugger in your bookmarks bar
  3. The debugger panel will appear on screen
  4. Interact with ads or page elements as normal — the tool will capture and display ad debug data in real time
The tool only runs on the page you're currently viewing. Navigate to a new page and click again to re-run it.

Need Help?

  • Tool not loading? Try refreshing the page first, then click again
  • Seeing a blank panel? The page may have blocked external scripts — try disabling your ad blocker
  • Still stuck? Contact Kamal
